Development Frameworks


Impact of Common Web Development Technologies and Frameworks

Framework

Frameworks are website development tools available online that make it quick and easy to create websites.

Frameworks have many pre-built functions reducing the need to write hours of code, which is more efficient, and easier for any type of user.

A widely used framework will have a big focus on security and the tool will also allow you to report security issues.

Most are free and open source reducing web development costs but also making it available to hackers that may study it and find ways to abuse it.

Impact of Common Web Development Technologies and Frameworks

Most frameworks are free, open source and secure. Frameworks contain a lot of built in files which can seem daunting to new users but for a big project it means the user does not have to write repetitive code making development faster. On the downside, these built in files can become corrupted if a user incorrectly or accidentally edits them. Templates and themes available within the frameworks allow users to create attractive, user friendly pages with very little input.

Foundation – Easily lets you customize pages, providing you with HTML, CSS and JavaScript coding. The foundation framework uses its own grid system making responsive design easy.

AngularJS – Open source ‘Model-View-Controller’ framework built on JavaScript. Used widely for creating dynamic single page applications that are easy to maintain.

Bootstrap – This is the most popular framework built on CSS. It also has its own grid system making it easy to develop mobile-first and responsive websites. Criticised for creating websites that are all a similar layout.

The core of a framework cannot be altered so design can be limited. Frameworks usually contain good user documentation and support forums for example to get useful help quickly.

Design

Website development involves designing the layout of your pages and structuring the content whilst considering your target audience. Is the content relevant and in a logical order? Is the site easy to find and easy to use? How will users navigate through your site? HTML is used to define the content and structure of web pages; CSS is used to style the pages e.g. font size and colour. Websites should always be tested before being uploaded to the Internet. There are many different web browsers and several different device screen sizes used to view web pages these days.


Rich Internet Application Frameworks

An RIA is a web application with the look, feel and features of a desktop application for example Google Maps, which you can create using software like the Google Web Toolkit, JavaScript Libraries, Adobe Flash Builder and Microsoft Silverlight.
