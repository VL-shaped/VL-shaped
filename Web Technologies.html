Website Technologies


Technologies, Tools and Software used to Develop Websites

Front end and Back end Technologies – Front end development manages the look and feel of a website/web app i.e. what the user (client) sees. Back end development manages the server side of an app/site i.e. how it works.

Back end Database Management Language SQL

Structured Query Language is a standard programming language designed for maintaining, accessing and manipulating databases. Any website collecting data or requiring information from a database (queries) will require the use of SQL. Most web hosting providers offer easy to use web applications to manage databases. Web site functionalities that require databases are online shopping facilities with real-time stock levels, and the storage of account registration details for example.

Front end Web Development Languages (client side)

JavaScript (JS) is used to add interactivity and functionality to web pages such as games and animations.

HTML (Hypertext Markup Language) is a standard set of syntax used to define the content and structure of web pages that will work on any computer system.

CSS (Cascading Style Sheets) is a language used to style the pages e.g. font size and colour.

Front end Web Development Frameworks

Bootstrap – Built in responsiveness makes sites look amazing on any device and browser. Certain elements can be as hidden on smaller devices and visible on bigger desktop pcs. Brilliant for a site that you expect most users will be viewing on their mobile phones. Bootstrap contains user documentation to help you learn the syntax and elements etc. Unfortunately the sheer amount of files and data within Bootstrap can cause slow loading times for web pages.

Foundation – CSS framework like Bootstrap but known best for its responsive menu. You will need to know more about CSS etc as the system for customising your pages is more complex than Bootstrap but can offer more unique looking sites.

AngularJS – JavaScript Framework that extends HTML allowing the creation of Rich Internet Applications. Another example is Semantic UI.

Web Application Languages for Back end (server side)

Ruby – Free, Open Source language that is dynamic with easy to learn syntax. It is flexible in that you can freely edit, copy, download and use it.

PHP – Widely used free, open source language. Easier to learn than other back end languages so often the first a back end programmer may learn. Used to collect form data on web pages.

Web Application Frameworks for Back end

Django is a Python framework used for large projects.

Flask is another Python based framework used for small projects.

Ruby on Rails is a Ruby framework mainly used for small projects. Another example is ASP.NET and Ruby On Rails.

Computer networks use Protocols as a set of rules that define how data travels through a network

They standardise the way that networks, and software/hardware from different manufacturers communicate, using a layered approach to describe different functions.

The Internet uses a four-layer model known as TCP/IP, created prior to the OSI model. Protocols send and receive data in what are called Packets.

Application Layer Provides access to network resources, applications and the user interface (FTP, SMTP, DHCP, HTTP)

Presentation Layer Encrypts, decrypts and compresses data including encryption (ASCII, SSL)

FTP (File Transfer Protocol) - Software using FTP can upload and download files to another computer.

HTTP (Hypertext Transfer Protocol) This Internet Communication Protocol is used by web browsers to access website images and pages etc from web servers all over the world.

HTTPS (Secure Hypertext Transfer Protocol) is a standard for transmitting web pages securely.
