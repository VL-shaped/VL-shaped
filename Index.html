Website Design and Development


Websites are used for marketing, customer information and services, business communications, online sales and shipping.

To Publish a website on the Internet;

Purchasing Web Space from a Web Hosting company (e.g. A2 Hosting). The files needed for your website to function will be stored on a server that runs 24 hours a day to give users constant access to your website.

Purchase a Domain Name from a Registrar (e.g. 123-Reg). Choose and check the availability of your preferred name. Some companies now offer both Domain Names and Web Hosting space as a package. If purchased from separate companies the domain name must be pointed at your separate web space using nameservers. This is relatively easy nowadays given the Dashboard Interface provided by Hosting companies.

Log in to your web space and upload your files, images and scripts to the server. There are now many ways to this for example, via a web server for example Microsoft IIS, an FTP client, or via the hosts website using a dashboard connecting you to your web space. These environments allow you to manage your files and folders with ease.

Register your website with search engines to be included in their search results. For example, Google is the most popular search engine in the world. The Google Site Verification tool quickly and easily adds your site to their index for free. You can do this manually via the Google Search Console. A Google account has many features that are very helpful for web developers for example you can register your business on Google using Google My Business.

What is a URL?

A Uniform Resource Locator is an address to a resource on the Internet for example; 'http://www.example.com/index.html'

It can be seen in your web browser when you are viewing a web page, at the top of the window in the 'address bar'

The Internet

The Internet is a global mesh network (large group of connected WANs) providing access to the world wide web (WWW). The WWW contains a vast amount of documents and resources that can be accessed via Hyperlinks (a word or image containing the address of the resource, to be clicked on). Hyperlinks are a standard technique using URLs (Uniform Resources Locator) to specify which resources from the internet they will connect to and which format it will be presented in. The Internet uses a standardised set of rules (Protocols) called TCP/IP (Transmission Control Protocol/Internet Protocol).
